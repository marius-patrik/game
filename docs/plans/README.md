# Plans

One plan per non-trivial feature, named `<issue>-<slug>.md`. Drafted by the overseer (or a planning agent) **before** an execution agent is dispatched.

Plans are living documents: `draft` → `in-progress` → `shipped`. After shipping, the **Retro** section captures what actually happened.

See [.claude/skills/planning/SKILL.md](../../.claude/skills/planning/SKILL.md) for the template and process.
